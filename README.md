# Agentic AI IT Support System

## Project Overview
AI-powered IT support system that automatically classifies 
and solves employee IT problems.

## Tech Stack
- n8n (Workflow Automation)
- Groq AI (LLaMA 3.3 70B)
- Python (Testing)
- HTML/CSS/JS (Web Portal)

## How it Works
1. Employee opens web portal
2. Types their IT problem
3. AI classifies and solves instantly
4. Returns category, priority, solution

## Project Structure
- it_support_portal.html → Web interface
- workflow.json → n8n workflow
- test_script.py → Python test script
